# MateriAlphabet
---

## Introduction

MateriAlphabet is a SVG library of material designed latin letters. It aims at creating a visual paper-like feeling, as if letters where folded ribons.

## Guidelines

Some guidelines for the project :
- **Dimensions**
  - Height: 160px
  - Width: 120px
  - Element width: 30/40px
- **Shadows**
  - Foreground pathes:
    - offset-x: 1px
    - offset-y: 1px
    - blur: 1px
    - spread: 1px
  - Background pathes:
    - offset-x: 0px
    - offset-y: 0px
    - blur: 1px
    - spread: 0px
- **Default colors**:
  - Foreground: #2196F3
  - Background: #0D47A1
